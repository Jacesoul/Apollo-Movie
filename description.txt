### Apollo 

- yarn add styled-components react-router-dom
- Apollo는 GraphQL API를 사용하기 좋은 방법이다.
- REST API는 URL로 가서 JSON을 받아오지만 GraphQL은 query를 보내거나 mutation을 보내야한다.
- npm install @apollo/client graphql
- apollo-boost는 GraphQL Yoga처럼 이미 모든걸 설정해둔 package이다.
- ... && ... && ... 의 의미는 앞의 두개가 true라면 마지막을 실행하라는 뜻이다.
- React Apollo에서 멋진점은 cache기능이 있다는것이다.

- local state란 API에서 넘어온 데이터를 바꿀수 있는것이다.
